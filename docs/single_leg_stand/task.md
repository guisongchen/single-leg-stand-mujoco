实验配置：MuJoCo 单腿站立

---
## 1. 实验目标

在 MuJoCo 仿真环境下，使用 G1 浮基人形机器人，逐步达成以下行为目标：

1. **双足站立稳定**：维持双足站立姿态不发散
2. **双足 → 单足过渡**：从双足平稳过渡到单足支撑，过程中不失稳
3. **单足支撑保持**：进入单足支撑后能够维持平衡一段时间

> 本文档只规定**目标行为**与**评估指标**，不规定具体的控制算法、状态机设计、求解器选择或参数取值。实现方可自行选择实现方式，重要的是最终行为满足第 5 节的评估指标。

---
## 2. 仿真环境（约束条件）

| 参数 | 设置 |
|------|------|
| 仿真器 | MuJoCo |
| 机器人模型 | `g1_23dof.xml` |
| 时间步长 | `dt_sim = 0.002 s` |
| 重力 | `[0, 0, -9.81]` |
| 地面 | MuJoCo 模型自带平面 |
| 摩擦系数 | `mu = 0.8` |

- 模型路径：`/home/ccc/projects/unitree_ros/robots/g1_description/g1_23dof.xml`
- 模型类型：浮基人形（MuJoCo `free joint` + 23 个驱动关节）

---
## 3. 机器人初始条件

机器人以稳定的初始站姿开始仿真：

- 双足平行落地
- 初始姿态在物理上保持稳定（不会立即发散）

注：MuJoCo 下完全直腿的零位姿态在物理上比较脆弱，因此初始姿态需要经过适当调整。具体的关节角度、damping、是否轻微屈膝等细节属于实现选择，不强制规定。

---
## 4. 任务阶段（行为目标分解）

任务从**行为层面**划分为以下三个阶段。具体由几个内部阶段实现、采用什么切换判据、由什么控制器执行，由实现方决定。

### 4.1 双足站立保持

机器人维持双足站立姿态稳定。

行为要求：
- 双脚都保持有效的法向接触力
- 足端不发生明显滑移
- 基座位姿不发散

### 4.2 双足 → 单足过渡

将负载从双足转移到单一支撑脚，并最终抬起摆动脚。

行为要求：
- 整个过渡过程中机器人不摔倒
- 摆动脚平稳脱离地面接触
- 支撑脚始终保持稳定接触
- CoM 顺利移动到支撑脚附近

### 4.3 单足支撑保持

在单足支撑状态下维持平衡一段时间。

行为要求：
- 支撑脚保持充分的法向接触力
- 摆动脚保持离地
- 基座位姿不发散
- 至少维持 3 秒以上

---
## 5. 评估指标

| 指标 | 目标值 |
|------|--------|
| 双足站立稳定性 | 不发散 |
| 双足足端滑移 | `< 0.005 m` |
| 双足阶段 CoM 位置 RMSE | `< 0.02 m` |
| 单足支撑稳定时间 | `>= 3 s` |
| 控制循环实时性 | 求解时间不超过仿真步长 |
